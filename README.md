# Digital-smart-card-system

A premium modern digital business card web application designed with a mobile-first experience, futuristic UI aesthetics, and interactive sharing features. Built to provide a professional online identity with seamless networking and contact-sharing capabilities.

---

## ✨ Features

- Premium dark glassmorphism UI
- Fully responsive mobile-first design
- WhatsApp direct chat integration
- QR code generation and sharing
- Save contact (.vcf download)
- Floating share hub
- Copy profile link functionality
- Custom WhatsApp/Open Graph preview
- Google Analytics integration
- Smooth page transitions
- Premium loading animation
- Interactive social media section
- Modern service cards with hover effects
- Professional location section with styled map
- Smooth animations and micro-interactions

---
